Java中有两种类型的数组：

- 基本数据类型数组；
- 对象数组；

当一个对象使用关键字“new”创建时，会在堆上分配内存空间，然后返回对象的引用，这对数组来说也是一样的，因为数组也是一个对象；栈中存放的是堆中对象实例化的地址引用
